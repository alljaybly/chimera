## Troubleshooting

### If backend won't start:
```bash
cd digital-phylactery
npm install
npm run dev:backend
```

### If frontend won't start:
```bash
cd digital-phylactery
npm install
npm run dev:frontend
```

### If database issues:
```bash
cd digital-phylactery
rm -f phylactery.db
npm run dev:backend  # Will recreate database
```
